# 🤖 Multimodal Local Chat Application

A privacy-first, locally hosted AI assistant that supports conversational interaction with PDFs, images, and more—powered by Streamlit and LangChain.

---

## 🚀 Getting Started

Follow these steps to set up and run the app on your machine.

### 1. Clone the Repository

```bash
git clone https://github.com/Madmax729/PDF-s-and-more-
cd PDF-s-and-more-
```

### 2. Init and run the code

```bash
#creates a virtual environment so you don't download the packages globally on your computer
python -m venv venv
#activate the environment on windows
venv\Scripts\activate

#or

#activate the env on linux/mac
source venv/bin/activate

#install the dependencies
pip install -r requirements.txt

#run the app
streamlit run app.py

#... 

# quit the virtual env when you're done
deactivate
```
